# Order Management App

A lightweight order management system that manages the core workflow of an online store, by processing customer purchases and tracking product stock

---
## Tech Stack

###Frontend
* **Framework:** React, Next.js
* **Styling:** Tailwind CSS

###Backend
* **Runtime:** Node.js (LTS)
* **Framework:** Express.js

###Database
* **Type:** PostgreSQL
* **ORM:** Prisma

###Cloud
* **Platform:** Microsoft Azure

---

## Project Structure

* `/frontend/` - Client-side user interface
* `/backend/` - Node.js API and business logic
* `/database/` - Database schemas, connection scripts, and seed data
* `/docs` - API documentation and system architecture details
